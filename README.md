# mojarras-frontend

Frontend simple para registro e inicio de sesion de usuarios.

## Requisitos

- Node.js 18+ (recomendado)
- npm
- Backend de mojarras-api corriendo en http://localhost:8080

## Como correrlo rapido

1. Abre terminal en la carpeta mojarras-frontend.
2. Instala dependencias:

   npm install

3. Inicia el proyecto:

   npm start

4. Abre en el navegador:

   http://localhost:3000

## Flujo de uso

1. Entrar a Registro y crear cuenta.
2. Ir a Login e iniciar sesion con ese usuario.
3. Si el login es correcto, te manda a Perfil.
